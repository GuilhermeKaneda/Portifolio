DESENVOLVIMENTO WEB

HyperText Markup Language (HTML)

HTML é o bloco de construção e marcação mais básico da web. Os marcadores, chamados de “tags”, servem para indicar a função de cada elemento na página, como textos, imagens e vídeos, além de suas conexões com outros elementos (links) e como eles serão interpretados pelo navegador.

Em nossas aulas, iniciamos com o desenvolvimento web simples, apenas com HTML, a fim de familiarizar-nos com tais funções.

Tag Função
<html> Define o documento como HTML.
<head> Contém metadados e links para scripts e estilos.
<title> Define o título do documento, exibido na aba do navegador.
<body> Contém o conteúdo visível da página HTML.
<h1> a <h6> Define cabeçalhos de diferentes níveis crescentemente.
<p> Define um parágrafo.
<a> Define um hyperlink.
<img> Define uma imagem.
<ul> Define uma lista não ordenada.
<ol> Define uma lista ordenada.
<li> Define um item de lista.
<table> Define uma tabela.
<div> Define uma seção ou um contêiner genérico.
<input> Define um campo de entrada em um formulário.
<button> Define um botão clicável.

Tags Semânticas

Tags semânticas no HTML são elementos que dão significado ao conteúdo, indicando ao navegador e aos desenvolvedores o tipo de informação que está sendo apresentada.

Tag Função
<header> Define o cabeçalho de uma seção ou página..
<nav> Indica uma seção de navegação dentro do documento.
<main> Representa o conteúdo principal do documento.
<article> Encapsula conteúdo independente e autônomo, como artigos.
<footer> Define o rodapé de uma seção ou página.

Document Object Model (DOM)

O DOM representa a estrutura do documento como uma árvore de objetos, onde cada nó da árvore corresponde a uma parte do documento. Essa estrutura permite que linguagens de script, como JavaScript, acessem e manipulam o conteúdo, a estrutura e o estilo de forma dinâmica.

Cascading Style Sheets (CSS)

CSS é uma linguagem de estilo utilizada para descrever a apresentação de documentos HTML. Define as cores, fontes, tamanhos, posicionamento e qualquer outro valor estético para os elementos da página.

Assim, as Folhas de Estilo em Cascatas são imprescindíveis, manipulando a forma e aparência dos elementos da sua página.

Estrutura do CSS

Em um arquivo CSS, toda regra possui:

  1. Seletor: seleciona um elemento HTML
  2. Propriedade: característica do estilo
  3. Valor: indica como estilizar a propriedade

JavaScript (JS)

JavaScript é uma linguagem de programação de alto nível, interpretada, usada principalmente para criar e controlar conteúdo dinâmico em páginas da web.

Ele pode manipular tanto o HTML quanto o CSS buscando os elementos através do Document Object Model (DOM), uma interface de programação que representa a estrutura de documentos HTML como uma árvore de objetos.

Funções Funcionamento Sintaxe
getElementById Seleciona um elemento pelo seu ID document.getElementById('ID')
getElementsByClassName Seleciona todos os elementos de uma classe document.getElementById('classe')
getElementsByTagName Seleciona todos os elementos de uma tag document.getElementsByTagName('tag')
querySelector Seleciona elementos usando seletores CSS document.querySelector('.minhaClasse')
